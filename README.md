# 📊 REA: Evaluación en la Era de la Inteligencia Artificial (2026)

¡Bienvenido/a al repositorio de este Recurso Educativo Abierto (REA)! Este material didáctico está diseñado para ofrecer a los docentes estrategias, marcos teóricos y herramientas prácticas para afrontar y rediseñar los procesos de evaluación formativa y sumativa en el panorama educativo actual de 2026.

El recurso se ha desarrollado por completo utilizando la herramienta de autor **eXeLearning (versión 4.0)**, lo que permite su libre distribución, edición y visualización como sitio web interactivo.

## 🛠️ ¿Cómo reutilizar o personalizar este recurso?
Este REA está pensado para ser adaptado por la comunidad docente. Si deseas modificarlo, el proceso es muy sencillo:

1.  **Descarga el archivo fuente:** Descarga el archivo con extensión `.elpx` alojado en este repositorio.
2.  **Abre eXeLearning:** Ejecuta [eXeLearning](https://exelearning.net/) (versión 4.0 o superior) en tu equipo y carga el archivo descargado.
3.  **Adapta el contenido:** Modifica los instrumentos de evaluación, añade tus propias rúbricas o ajusta las actividades según el nivel de tus alumnos.
4.  **Exporta y publica:** Vuelve a exportarlo como "Sitio web" (Carpeta autocontenida) para subirlo a tu propio GitHub, blog o entorno virtual de aprendizaje (Moodle, Google Classroom, EVAs).
